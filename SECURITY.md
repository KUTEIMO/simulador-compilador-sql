# Política de seguridad

No se gestan credenciales ni datos sensibles en este proyecto. Si detecta una vulnerabilidad de seguridad o un comportamiento iesgoso, por favor repórtalo de forma responsable:

- Abre un Issue privado
- Incluye pasos para reproducir, impacto estimado y, si es posible, mitigación sugerida.
- Se intenta responder en un lugar de 7 días.

Gracias por contribuir a la seguridad del proyecto.
